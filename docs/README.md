## Overview

OpenSite Analytics is a self-hosted analytics platform that provides:

- **Privacy-First**: Your data stays on your own server
- **Real-Time Analytics**: Track page views, events, and sessions in real-time
- **Multi-Domain Support**: Monitor multiple websites from one dashboard
- **Easy Integration**: Simple JavaScript SDK similar to Google Analytics
- **Rich Features**: Custom events, goal tracking, UTM parameters, and more

## Getting Started

### Quick Start

1. Clone the repository
2. Follow the [Installation Guide](INSTALLATION.md)
3. Login to the dashboard
4. Add your first site
5. Copy the tracking code to your website
