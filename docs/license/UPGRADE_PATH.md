# License Upgrade Path

This guide explains how to upgrade from one license type to another.

## Upgrade Options

### MIT → Commercial

**Reason:** Need commercial use without attribution

**Process:**
1. Purchase Commercial License
2. Remove attribution from product
3. Receive license key
4. Update documentation

**Cost:** $499/year (prorated if upgrading mid-year)

### MIT → Enterprise

**Reason:** Need enterprise features and support

**Process:**
1. Purchase Enterprise License
2. Remove attribution from product
3. Receive license key and account manager
4. Schedule onboarding

**Cost:** $2,499/year (prorated if upgrading mid-year)

### Commercial → Enterprise

**Reason:** Need priority support and custom features

**Process:**
1. Contact sales for upgrade quote
2. Receive prorated invoice
3. Pay difference
4. Receive Enterprise benefits

**Cost:** Difference between licenses (prorated)

### Commercial → OEM

**Reason:** Need to embed in product

**Process:**
1. Purchase OEM License
2. Receive OEM license key
3. Add "Powered by OpenSite Analytics" attribution
4. Start distribution

**Cost:** $9,999/year (prorated if upgrading mid-year)

### Enterprise → White Label

**Reason:** Need rebranding and resale rights

**Process:**
1. Purchase White Label License
2. Receive White Label license key
3. Begin rebranding process
4. Update legal agreements

**Cost:** $19,999/year (prorated if upgrading mid-year)

## Downgrade Options

### Enterprise → Commercial

**Reason:** No longer need enterprise features

**Process:**
1. Contact sales
2. Receive credit for remaining term
3. Downgrade to Commercial
4. Lose enterprise features

**Cost:** Credit for difference

### OEM → Commercial

**Reason:** No longer distributing

**Process:**
1. Contact sales
2. Stop distribution
3. Downgrade to Commercial
4. Maintain existing deployments

**Cost:** Credit for difference

## Cancellation

### MIT License

No cancellation needed - perpetual use.

### Commercial/Enterprise/OEM/White Label

**Process:**
1. Provide 30 days notice
2. Stop new deployments/distribution
3. Existing deployments continue until term end
4. No refund for remaining term

**Exception:** Licensor may cancel for material breach with immediate effect.

## Migration Steps

### Step 1: Purchase New License

Contact sales@opensite-analytics.com with:
- Current license type
- Desired license type
- Reason for upgrade
- Company details

### Step 2: Receive New License

- New license key
- Updated terms
- Prorated invoice (if applicable)

### Step 3: Update Product

- Install new license key
- Update attribution (if required)
- Update documentation
- Inform team of changes

### Step 4: Verify Compliance

- Ensure new license terms are met
- Remove old attribution (if applicable)
- Add required attribution (if applicable)

### Step 5: Transition Period

- 30-day transition period
- Support during transition
- Account manager assistance (Enterprise/White Label)

## Refund Policy

### No Refunds

All license sales are final. No refunds will be provided.

### Credits

Credits may be provided for:
- Upgrades to higher-tier licenses
- Downgrades within 30 days of purchase
- Service failures (SLA violations)
