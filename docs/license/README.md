## FAQ

**Q: Can I use MIT license for commercial projects?**
A: Yes, but attribution is required.

**Q: Can I remove attribution with MIT license?**
A: No, attribution is required for MIT license.

**Q: What happens if my commercial license expires?**
A: You must renew or cease commercial use.

**Q: Can I upgrade my license?**
A: Yes, contact sales for upgrade pricing.

**Q: Can I get a refund?**
A: No, all sales are final.

**Q: Do licenses include source code access?**
A: Yes, all licenses include source code access.

**Q: Can I modify source code?**
A: Only with White Label license.

**Q: Do I need a license for internal use?**
A: No, MIT license covers internal use.

**Q: Can I use in open source projects?**
A: Yes, with MIT license.
